# window 命令

## 返回上一级目录：
```hash
cd..
```
## 进入下级目录：
```hash
cd xx // xx：文件名
```

## ping ip地址
```hash
ping 10.23.32.111
```
ping命令可以通过向目标网站发送数据包，从数据包的平均达到时间和丢包率来判断，从本地到目标网站的网络情况

## 创建文件
```hash
mkdir x // xx：文件名
```
